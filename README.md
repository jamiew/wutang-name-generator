# Wu-Tang Names Forever

Wu-Tang is for the children. And the children need names.

## The Origin Story

This project started by scraping the classic [Wu-Tang Name Generator](http://www.mess.be/inickgenwuname.php) â€” the same one that blessed Donald Glover with "Childish Gambino." We ripped all 676 possible two-letter seed combinations, parsed out the words, and built our own local generator.

But we didn't stop there.

## The Expansion

We've been building out the word lists with cyber/hacker/sci-fi flavor. Think less "Ghostface Killah" and more "Glitched_Netrunner". The vibe is Wu-Tang meets cyberpunk meets vaporwave meets whatever sounds hard.

Current word counts:
- **229 first-words** (adjectives/descriptors)
- **230 second-words** (nouns/titles)

That's **52,670 possible combinations**.

## Usage

```bash
./wutang        # one name
./wutang -n 10  # ten names
```

## Growing the Lists

Just edit `first-words.txt` or `second-words.txt`. One word per line. The script picks em up automatically.

## The Originals

`originals.txt` has the real Wu-Tang Clan members and affiliates for reference. RZA, GZA, ODB, Method Man, Raekwon, Ghostface, Inspectah Deck, U-God, Masta Killa, Cappadonna, and the whole extended universe.

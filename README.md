# Wutang Names Forever

This script fetches lots of random names from the OG [mess.be/inickgenwuname.php](https://www.mess.be/inickgenwuname.php) in an attempt to duplicate their excellent word list

All praise to the original <3

## Usage

```sh
./fetch-words.sh
./parse-words.sh
```

Now you have names.txt, first_words.txt, and second_words.txt
